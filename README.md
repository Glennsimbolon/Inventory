# 🏪 Aplikasi Inventory Realtime

Aplikasi manajemen inventaris berbasis mobile menggunakan **Flutter** dan **Supabase** sebagai backend. Mendukung sinkronisasi data real-time antar perangkat.

## ✨ Fitur Utama

- 📊 Dashboard dengan grafik pergerakan stok
- 🏢 Manajemen Supplier (CRUD)
- 📦 Manajemen Barang dengan foto
- 📥 Pencatatan Barang Masuk (auto update stok)
- 📤 Pencatatan Barang Keluar (dengan validasi stok)
- 📄 Ekspor Laporan PDF & Excel
- 🔄 Sinkronisasi Real-time

## 🛠️ Teknologi yang Digunakan

- **Flutter** - UI Framework
- **Supabase** - Backend (PostgreSQL, Auth, Storage, Realtime)
- **Dart** - Bahasa Pemrograman
